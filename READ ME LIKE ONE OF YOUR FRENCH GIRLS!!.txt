HDSD gấc đơn giản bé iu của mình
1. Down tất cả các file này bằng cách:
- Trong mục Code màu xanh lá, chọn mũi tên
- Chọn Download ZIP
2. Giải nén
2. Double click file "index" là xong

P/S: 
- Đừng bấm những cái khác, kể cả thư mục ingre 
(thư mục này có gì là trong file index có hết nên bấm vô mất zui ráng chịu)
- Chụt bạn
